# Market Insights Hub

## Description
**Market Insights Hub** is a bilingual (Amharic & English) web application designed to provide timely news, market analysis, and interactive charts for the **crypto** and **forex** markets in Ethiopia and across Africa.  
Built with **Next.js**, **React**, and **Tailwind CSS**, the platform delivers a seamless and responsive experience for traders, investors, and financial enthusiasts who want to stay ahead in rapidly changing markets.  

Key features include **real-time news updates**, **interactive financial charts**, and **language toggle support** to cater to both local and international audiences.

---

## Features
- 🌐 **Bilingual (Amharic & English) User Interface**  
- 📰 **Real-time Crypto and Forex News Feed**  
- 📊 **Interactive Price, RSI, and Volume Charts**  
- 📱 **Responsive Design for all devices**  
- ⚡ **Fast Navigation and Optimized Performance**  
- 🔍 **Search and Filter for Market Data**

---

## Tech Stack
- **Next.js** – Framework for server-side rendering and static site generation
- **React** – UI library for building interactive components
- **Tailwind CSS** – Utility-first CSS framework for styling
- **Chart.js** – Data visualization library for charts
- **Axios** – HTTP client for API requests
- **Vercel** – Deployment and hosting platform
